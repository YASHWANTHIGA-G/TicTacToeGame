# 🎮 Dark Mode Tic Tac Toe

A modern **Tic Tac Toe game** built using **HTML, CSS, and JavaScript** with a dark-themed gaming UI and sound effects.

## ✨ Features
- 🌙 Dark mode gaming interface
- 🎯 Two-player gameplay
- 🏆 Winning detection
- 🤝 Draw detection
- 🔊 Winning sound effect
- 🎵 Draw sound effect
- 🔄 Restart game option
- 📱 Responsive design

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript

## 🚀 Live Demo
🔗 https://tic-tac-toe-game-rose-alpha.vercel.app/

## 🎮 How to Play
1. Player **X** starts first.
2. Players take turns placing **X** and **O**.
3. First player to align 3 symbols wins.
4. If all cells are filled and no winner, it's a draw.

## 🔊 Sound Effects
- **win.wav** → Plays when a player wins
- **lose.wav** → Plays when the match ends in a draw
